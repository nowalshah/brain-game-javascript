# Brain Game Web Application

An interactive browser-based JavaScript application featuring puzzle challenges and a memory matching game designed to strengthen logical thinking and front-end development skills.

## Features
- Fetches puzzles dynamically from external APIs
- Implements async/await with fallback logic if APIs fail
- Stores user score and attempts using localStorage
- Includes a memory matching mini-game
- Built using modular ES6 JavaScript architecture

## Technologies Used
JavaScript (ES6), HTML5, CSS3, REST APIs, LocalStorage

## What I Learned
This project helped me practice:
- handling asynchronous API requests
- structuring modular JavaScript applications
- debugging real-world browser issues
- managing persistent user data

## How to Run
1. Clone or download the repository
2. Open index.html in your browser
