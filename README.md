# Brain Game Web Application

Interactive JavaScript web application featuring puzzle challenges and a memory matching game.

## Features

- Fetches puzzles dynamically from external APIs  
- Uses async/await with fallback logic if APIs fail  
- Stores score and attempts using localStorage  
- Includes memory matching mini-game  
- Modular JavaScript architecture (ES6 modules)

## Technologies Used

JavaScript (ES6), HTML5, CSS3, REST APIs, LocalStorage

## How to Run

1. Download or clone the repository
2. Open index.html in your browser
